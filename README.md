# 🚀 GitHub Radar 变现机会评估系统

这是一个专为**独立开发者**、**自媒体创作者**和**技术淘金者**打造的全栈式开源项目追踪与自动化变现运营系统。它能帮您第一时间捕获全网最火的开源热点，量化海内外“信息差红利”，并利用最新 AI 技术（DeepSeek + 火山引擎）全自动将其包装为高转化率的自媒体爆款内容。

---

## 🌟 系统六大核心板块功能

本系统的界面左侧栏严格按照“发现 -> 挖掘 -> 运营 -> 调研 -> 配置”的工作流设计，共分为六大核心板块：

### 📌 1. 变现情报中枢 (Hub)
一站式发现信息差红利，智能评估变现潜力的“雷达大脑”。
- **社媒热力雷达与比对中心**：系统每 6 小时自动抓取全球社交媒体（如 YouTube 技术测评视频、Reddit 副业热榜与少数派爆款工具等）。
- **信息差指数计算**：利用 AI 深度比对国内外的讨论热度，自动计算出该项目的“GitHub 热度”、“海外热度”、“国内热度”，最终量化出一个 **[信息差指数]**。指数越高的项目，在国内做教程或商业化的红利极大（强推选品）。
- **一键快捷流**：对看中的强推变现选品，可以直接点击“发文排版”或“复制文案”进行快速转化。

### 📌 2. 榜单挖掘 (Trending)
实时追踪最火的底层开源核心库与项目源码：
- **多语言热点追踪**：系统自动从 GitHub Trending 抓取 JavaScript、Python、Go、Rust 等热门语言的最新高星项目。
- **动态筛选**：支持通过“历史总星标多寡”与“今日增长暴涨榜”进行交叉筛选，防止错过任何底层技术风口。
- **一键打包助手**：内置自动化打包功能，可快速将看中的开源项目提取并打包成适合发给小白用户的“一键免安装整合包/本地运行包”，大幅简化变现资源的分发流程。

### 📌 3. 自媒体运营 (Operations)
系统的核心自动化文案生成工厂。全网首创的**“一鱼两吃”工作流**，一键将枯燥的代码转换为现象级图文：
- **红狐真实爆款注入**：在写文案前，系统会调用红狐（Redfox）API 实时抓取对应细分赛道的小红书 1000+ 赞、公众号 10w+ 阅读的真实验证爆文作为模仿底料。
- **🌋 火山多模态视觉分析**：自动拉取微信真实验证爆文的封面图，调用火山引擎视觉大模型 (`Doubao-vision-pro-32k`) 深度拆解封面的视觉刺激点与排版逻辑。
- **🐳 DeepSeek 爆款生成器**：根据抓取的数据，DeepSeek 模型会同步生成“小红书诱饵文案”与“公众号深度长文”（附带多种不同情绪风格的公众号爆款标题供挑选）。
- **🎨 智能生图辅助**：自动为您提炼可直接发送给即梦（Seedream）或 Midjourney 的高保真封面插画提示词。
- **☁️ 智能飞书台账同步**：一键同步生成的爆款文案至飞书多维表。自动提取文章标题、平台属性并记录发布状态，极大方便内容资产的管理与发布排期。


### 📌 4. 小红书探测 (Xiaohongshu Data)
免除流量玄学的“数据天眼”，供您手动进行深度市场调研：
- **精准赛道检索**：输入任意泛关键词（如：AI工具、自媒体运营），穿透平台数据壁垒，直接呈现全网最新互动量 1000+ 的超级爆款笔记大盘。

### 📌 5. 公众号大盘 (WeChat Data)
- **流量天花板探测**：默认近 7 天数据自动回溯（不足 10 条自动扩大至 30 天）。搜索赛道词，让您在自己构思内容时，能直接扒出微信公众号里的现象级流量密码作为对标。

### 📌 6. 系统配置 (Settings)
- 集中管理大模型 API Keys 与第三方数据源秘钥（如 DeepSeek、火山引擎、红狐 API 等），支持即时保存生效。

## 🛠️ 技术栈与底层驱动

- **前端面板**: Vanilla JS + 现代原生 CSS
- **后端服务**: Node.js + Express.js + SQLite3 (数据持久化)
- **AI 智能中枢**:
  - **文本推理**: DeepSeek API
  - **多模态读图**: 火山引擎 (Doubao-vision-pro-32k)
  - **AI 绘画**: 火山引擎 (Doubao-seedream)
- **数据爬虫引擎**: 红狐 API (Redfox) 与 GitHub 实时 API

## ⚙️ 快速开始

### 1. 基础安装
```bash
git clone <repo-url>
cd <repo-folder>
npm install
```

### 2. 配置环境变量 (.env)
在项目根目录创建一个 `.env` 文件，并填入您的授权秘钥：

```env
# GitHub 访问凭证
GITHUB_TOKEN=your_github_personal_access_token

# 文本大模型
DEEPSEEK_API_KEY=your_deepseek_api_key

# 视觉大模型与图片生成
VOLC_API_KEY=your_volcengine_api_key

# 爆款数据抓取
REDFOX_API_KEY=your_redfox_api_key

# 您的公众号引流名称
WECHAT_ACCOUNT_NAME=YourWechatName

# 飞书台账多维表推送配置 (可选，基于 lark-cli)
FEISHU_LEDGER_BASE_TOKEN=your_feishu_bitable_base_token
FEISHU_LEDGER_TABLE_ID=your_feishu_bitable_table_id
```
> **💡 火山引擎特别提示**：火山引擎鉴权需要在代码 `src/classify/aiAnalyst.js` 头部将 `VOLC_VISION_MODEL` 和 `VOLC_IMAGE_MODEL` 替换为您在方舟控制台申请的 `ep-` 专属接入点 ID。

### 3. 一键启动
```bash
npm start
# 服务启动后，浏览器将自动打开监控台：http://localhost:3000
```
